# Introduction
 
Welcome to my Next.js + Supabase demo site!
I built this site to demonstrate a full-stack integration between a Next.js frontend and a Supabase backend. It showcases:

- A personality quiz that matches you to an LLM (Claude, Gemini, or ChatGPT) based on your answers
- Supabase integration that saves quiz results to a PostgreSQL database in real time
- Row Level Security (RSL) policies that control who can read and write to the database
